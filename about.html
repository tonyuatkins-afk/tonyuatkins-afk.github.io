About

Tony Atkins is an electronics builder based in the Pacific Northwest.

The day job is in cybersecurity. That means spending years thinking about what makes systems trustworthy and where they fall apart, and developing a deep appreciation for secure design along the way. Outside work, the pull is vintage computing hardware. Barely Booting is where those two interests connect: bringing modern cryptography and properly engineered networking to machines that were built long before TLS existed.

It started with a question: could I turn my 286 into a daily driver? Not a novelty, not a retro nostalgia prop — an actual machine you could send email from, chat with friends on, browse the web with, and get real work done on. The honest answer in 2026 is “no, not without building a lot of new stuff.” So that’s what Barely Booting is: the lot of new stuff.

The flagship is NetISA, an ISA expansion card that gives pre-Pentium PCs a working path to the modern internet — WiFi and TLS 1.3 via an ESP32-S3 crypto coprocessor, bridged to the ISA bus by a CPLD. On top of the card is a growing software stack: Cathode (a text-mode web browser, sort of), Discord v2 for DOS, a Claude AI chat client, and the NetISA launcher. Alongside it, TAKEOVER is an interactive AI takeover simulator for DOS — no NetISA required, pure 8088 real mode. The common thread is making old computers do new things, properly engineered.

Everything is open source. Schematics, firmware, CPLD logic, DOS source, documentation, build logs, test suites, quality-gate results. The goal is for anyone to be able to build one.
