# barelybooting.com

Source for [barelybooting.com](https://barelybooting.com/), the project site for Barely Booting and [NetISA](https://github.com/tonyuatkins-afk/NetISA).

Static HTML + CSS, hosted on GitHub Pages. No build step, no frameworks.

## Pages

- **[Home](https://barelybooting.com/)**: project overview, screenshots, community links
- **[NetISA](https://barelybooting.com/netisa.html)**: specs, screenshots, architecture, firmware details
- **[Build log](https://barelybooting.com/log.html)**: reverse-chronological milestones
- **[About](https://barelybooting.com/about.html)**: background and contact
- **[RSS feed](https://barelybooting.com/feed.xml)**: build log updates
